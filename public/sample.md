# 《烈日焚城：我有移动冰窖，谁求都不给》

> **类型**：男频末日重生短剧  
> **题材**：末日求生 / 赘婿逆袭 / 重生 / 天灾  
> **受众**：25-35岁都市男性  
> **核心爽点**：赘婿复仇 + 资源垄断 + 权力反转

---

## 一、故事梗概

### 一句话故事
受尽羞辱的上门女婿重生极热末日前，掌控永冻空间，看岳家在烈日下跪地求冰。

### 故事主线
上门女婿苏寒在豪门岳家备受冷眼。前世，太阳暴致全球气温飙至70°C，岳家夺走他的冷库资源将他扫地出门，苏寒和妻子林婉在烈日下渴死。

重生回末日前一周，觉醒永冻空间系统，可储存物资并恒温零度。他拒绝岳家资源掠夺，买下工业冷藏车改装移动冰窖，加装太阳能反射镜面装甲和自卫武器。

末日爆发，地表成焦土，岳家豪宅变烤炉。曾经高傲的岳母带女儿上门求冰水。苏寒坐在冰窖中喝冰镇饮料，冷眼看前世背叛者在烈日下哀求。他驾车穿越废土建立冰窖避难所，用清凉资源交换忠诚，成为极热末世唯一的"冰霜之王"。

## 三、世界设定

### 时间背景
- **末日爆发**：公元2025年夏至次日
- **时间线**：末日前7天倒计时 → 末日后一年
- **灾难特征**：太阳风暴撕裂大气层，全球气温飙升至70°C

### 地点设定
- **烬城/烈日城**：中国东南沿海某二线城市
- **原人口**：300万 → **幸存者**：不足5万
- **关键区域**：
  - 城南别墅区（岳家豪宅）
  - 城北工业区（废弃冷链仓库）
  - 地下避难所网络（赵天豪控制）

### 末日法则
| 要素 | 设定 |
|------|------|
| 价值体系 | 一瓶500ml冰水 = 一条人命 |
| 温度阶级 | 25°C恒温区=贵族；能喝冰水=中产；地表求生=贱民 |
| 系统规则 | 永冻空间恒定零下20°C；资源只进不出自动保鲜；背叛者无法进入庇护范围 |
| 交通 | 改装装甲车、苏寒的移动冰窖 |
| 通信 | 对讲机（5公里）、信使、短波电台 |

### 第2集 觉醒

**【本集看点】**

---

#### 场次 2-1 岳家别墅·苏寒卧室 / 日 / 内

**人物**：苏寒

**【动作】**
苏寒猛地从床上坐起，大口喘气，胸口剧烈起伏。

他伸手摸嘴唇——皮肤完整，没有裂口。

翻开手掌——有血色的正常手，不是前世那双干枯的手。

空调冷风吹来，窗外传来蝉鸣声。

苏寒扭头看墙上的日历：**2025年6月15日**。

**苏寒（OS）**：距离太阳风暴爆发，还有七天……我真的重生了。

**【动作】**
苏寒翻身下床，冲到镜子前。

镜子里的人脸色蜡黄，低着头，肩膀习惯性缩着——一副卑微模样。

**【特效字幕】**
一行蓝色半透明字符在镜子里浮现。

**系统（VO）**：永冻空间已开启，当前容量十立方米，储存物资可恒温零下二十度。

**系统（VO）**：空间升级方式：宿主每承受一次羞辱，空间扩大十立方米。

**【动作】**
苏寒盯着镜中的自己，嘴角开始上扬，笑容越来越大，肩膀颤抖，但没有发出声音。

**苏寒（OS）**：承受羞辱？呵，过去三年我受的羞辱还少吗？岳家，你们就是我最好的升级材料。

苏寒深吸一口气，挺直腰背，抬起头。

**【音效】**
门外传来脚步声，门锁被粗暴转动。

门被推开，岳母周美玲站在门口，双手抱胸，下巴抬得很高。

#### 场次 2-2 岳家别墅·苏寒卧室 / 日 / 内

**人物**：苏寒、岳母周美玲

**岳母**：苏寒，你个废物还赖在床上？我儿子的房款你到底什么时候给我！

**【动作】**
岳母扬起右手，手掌对准苏寒的脸。

苏寒直视着那只手，一动不动。

**苏寒（OS）**：前世这一巴掌，我接了，然后跪下道歉，把银行卡双手奉上……这一世——

岳母的手掌在空中停顿。

**岳母（尖声）**：你今天是想挨打是吧？

苏寒嘴角勾起。

**苏寒**：我的钱，凭什么给你儿子买房？你儿子手脚健全，自己不会挣？

**岳母**：你……你敢这么跟我说话？！

### 第3集 反击

**【本集看点】**
- 正面冲突 → 系统升级 → 离开岳家

---

#### 场次 3-1 岳家别墅·客厅 / 日 / 内

**人物**：苏寒、岳母周美玲、林婉、林峰

**【动作】**
岳母的手掌挥下来。

苏寒抬手，五指握住岳母的手腕。

岳母瞪大眼睛，用力想抽回手，抽不动。

**岳母（尖叫）**：你……你敢抓我？！

**【动作】**
客厅里，林婉坐在沙发上玩手机，林峰抖着腿看电视。

两人听到岳母的尖叫，同时抬头。

苏寒松开岳母的手，从她身边走过，进入客厅。

岳母捂着手腕跟进来，手指颤抖着指向苏寒。

**岳母**：反了！真是反了天了！我们林家养了你三年，养出个白眼狼来！

**【动作】**
林峰从沙发上跳起来，冲过来指着苏寒鼻子。

**林峰**：你个吃软饭的，敢对我妈动手？信不信我打断你的腿！

林峰双手推在苏寒胸口。

苏寒后退两步，面无表情看着林峰。

**【特效字幕】**
一行系统提示在苏寒眼前浮现。

**系统（VO）**：检测到恶意攻击，空间增加十立方米。

**【动作】**
林婉放下手机，声音软软地开口。

**林婉**：苏寒，你别跟妈犟了，道个歉吧，妈也是为了咱们家好。

苏寒扭头看向林婉。

林婉低着头，手指搅着衣角，不敢看他。

**苏寒（OS）**：前世，就是这句话，让我跪下道歉，交出所有积蓄，然后末日来临时，你们把我推出门外。

**【特效字幕】**
**系统（VO）**：检测到极致羞辱，来自宿主妻子，空间增加二十立方米。

苏寒收回目光，看向岳母。

**岳母**：把钱交出来！不然今天别想走出这个门！

**【动作】**
林峰又推了苏寒一下。

**林峰**：听见没有，把钱交出来！老子要买房，你个废物就该出钱！

**【特效字幕】**
系统提示不断弹出，苏寒眼前的数字飞速跳动。

**系统（VO）**：当前空间容量——六十立方米，七十立方米，八十立方米……一百立方米。

**【动作】**
苏寒整理了一下被推乱的衣领，转身走向大门。

**岳母（尖叫）**：你去哪！给老娘回来！

**人物**：苏寒、彩票店老板

#### 场次 5-2 银行门口 / 日 / 外

**人物**：苏寒

**【动作】**
苏寒站在银行门口，手机震动。

苏寒盯着那串零，面无表情。

**苏寒（OS）**：前世，我用父母留下的三十万改装了冷藏车，被岳家抢走。这一世，我要用四百万打造一个移动冰窖，让所有人都求着我。

苏寒划开通讯录，拨出一个号码。

**电话（VO）**：喂，您好，这里是天润物流。
