这是一个 Next.js 14 + ReactFlow + Next-Intl + Tailwind + ShadCN/UI 生产级本地项目(没有后端API，不用Electron)：
项目需求：
1、新建项目，导入剧本skill，资产skill、分镜skill、生视频skill,每个项目创建一个资源生成的文件夹
2、新增文本节点，让用户输入命令调用各种工具去实现导入的各种skill;但需要加上skill文件的地址，创建资源的地址
3、轮循结果，展示生成的节点信息
4、图和视频保存在项目文件夹
5、使用next.js +reactflow
6、需要把用户输入的命令通过终端给各种agent 例如opencode openclaw等等
