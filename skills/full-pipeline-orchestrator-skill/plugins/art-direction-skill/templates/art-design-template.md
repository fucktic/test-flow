# 服化道设计产出模板

## 输出文件

- `global-assets/character-prompts.md`
- `global-assets/scene-prompts.md`

规则：

- **`global-assets/`**：跨集累积主真相源；本集相关人物 / 场景用 **章节标题（如「ep-01 新增」）** 或 **ep 前缀** 与跨集条目区分，**不**使用 `ep-xx/episode-assets/` 等快照路径。

## 人物提示词格式

```markdown
# 人物提示词

## [人物名]（ep-01 新增）

**出图要求**：一张图，左半边面部特写，右半边全身正面、侧面、背面设定图，白色背景。**画风须与本集场景 3×3 宫格共用下方「视觉规范」中的线条、渲染、色板与材质，全片人物与场景视觉一致。**

**视觉规范**（须与对应场景宫格块逐字一致或明确写「同场景宫格」并指向同一段落）：
整体风格：[描述]
色彩基调：[描述]
材质质感：[描述]

**提示词**：
[完整叙事描述式提示词]
```

## 场景提示词格式

```markdown
# 场景提示词

## ep-01 场景宫格

请生成一张 **3×3 九宫格**布局的电影场景环境图像（**仅此格数**，禁止 3×4、4×4）。每个格子代表一个独立场景，**所有 9 格必须保持同一视觉风格、同一渲染与色板**。若本集独立场景少于 9 个，余格写「留空」或纯氛围延续占位；若多于 9 个，须拆第二张 3×3 宫格（见 `art-direction-skill` 溢出策略）。

### 视觉规范
整体风格：[描述]
色彩基调：[描述]
材质质感：[描述]

### Panel Breakdown
格1——【[场景名]】
视角：[完整叙事描述]
重点：[核心视觉元素]
```

## 输出规范

- 中文提示词
- 复用项不重复写入全局主文件（可在全局文件中引用已有章节）
- 人物与场景 **视觉规范** 必须对齐，保证出图画风一致
